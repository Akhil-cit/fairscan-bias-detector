# FairScan — AI Bias Detector

AI-powered web app that detects bias in automated decisions using Gemini AI.

## What it does
Paste any decision text (loan rejection, hiring filter, content policy) → 
Gemini analyzes it → Returns bias score (0-100), bias types, explanation, and a fairer rewrite.

## How to run
Just open `FairScan_Prototype.html` in any browser. No install needed.

## Tech Stack
- HTML / CSS / JavaScript
- Gemini API (claude-sonnet-4-20250514)
